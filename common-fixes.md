Common Fix Cheat Sheet
Quick reference notes for busy support environments. These are the fast, reliable fixes used daily in Microsoft 365 and Windows support.

Outlook
Restart Outlook
Create a new Outlook profile
Turn Cached Mode off and on
Repair the mailbox
Clear Windows Credential Manager
Disable problematic add ins
Rebuild the search index

OneDrive
Restart OneDrive
Unlink and re link account
Run OneDrive reset
Check for unsupported file names
Pause and resume sync
Close Office apps blocking sync
Add shared folder shortcut in OneDrive online

Teams
Clear Teams cache
Restart Teams
Check audio and video device settings
Sign out and sign back in
Update Teams
Check if Outlook and Teams use the same account

Identity and MFA
Reset MFA in Azure AD
Unlock account
Clear saved credentials
Check sign in logs
Confirm licence assignment
Sign user out of all sessions

Windows and Network
Restart device
Flush DNS
Reset network adapter
Clear temporary files
Check date and time settings
Restart Print Spooler

PowerShell
Connect to Microsoft Graph
List user licences
Check sign in logs
List stale devices
Reset MFA methods

When to use this cheat sheet
These steps are ideal when
A user is waiting
You need a fast first response
You want to rule out common causes
You are triaging multiple tickets
You want to confirm whether the issue is local or cloud based

Notes
This cheat sheet is not a replacement for full troubleshooting.
It is a quick way to eliminate the most common causes before moving into deeper investigation.
