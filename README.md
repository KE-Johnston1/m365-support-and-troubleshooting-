Overview
This project brings together common Microsoft 365 issues and the steps used to troubleshoot them.It covers Outlook, OneDrive, Teams, identity, MFA, and device related problems. Everything is written in clear, simple language to reflect the way support is handled in real environments.

The aim is to build confidence, strengthen troubleshooting habits, and show how to communicate solutions in a way users understand.

Folder Structure
outlook
Mail flow issues, profile problems, search faults, shared mailboxes, and authentication prompts.

onedrive
Sync failures, Known Folder Move issues, SharePoint library sync, and common user mistakes.

teams
Audio problems, presence issues, calendar sync, browser behaviour, and client troubleshooting.

identity
MFA resets, account lockouts, Conditional Access blocks, legacy authentication, and licence related access problems.

powershell
Microsoft Graph PowerShell examples for identity, devices, sign in logs, and MFA checks.

support notes
Escalation guidance, service health checks, licensing checks, and common user errors.

common fixes
A quick reference sheet for fast triage in busy support environments.

User Facing Guides
Short, clear instructions written for users who need simple steps to fix common issues.

Reset your Microsoft 365 password
Go to https://aka.ms/sspr
Enter your email
Choose a verification method
Complete the security check
Create a new password
Sign out and sign back in

Fix OneDrive when files are not syncing
Click the OneDrive cloud icon
Check for warnings
Select View online
Restart OneDrive
If needed, unlink and sign in again

Join a Teams meeting
Open the invite
Click Join
Choose audio and video settings
Select Join now
Allow microphone and camera if prompted

Clear Teams cache on Windows
Close Teams
Press Windows key and R
Type %appdata%\Microsoft\Teams
Delete the folder contents
Reopen Teams

Fix Outlook when it will not open
Press Windows key and R
Type outlook.exe  /safe
Disable add ins
Restart Outlook normally
