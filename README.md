User Facing Guides
These guides help users fix common issues themselves and demonstrate clear communication skills.

Reset your Microsoft 365 password
Go to https://aka.ms/sspr
Enter your email
Choose a verification method
Complete the security check
Create a new password
Sign out and sign back in

Fix OneDrive when files are not syncing
Click the OneDrive cloud icon
Check for warnings
Select View online
Restart OneDrive
If needed, unlink and sign in again

Join a Teams meeting
Open the invite
Click Join
Choose audio and video settings
Select Join now
Allow microphone and camera if prompted

Clear Teams cache (Windows)
Close Teams
Press Windows key + R
Type %appdata%\Microsoft\Teams
Delete the folder contents
Reopen Teams

Fix Outlook when it will not open
Press Windows key + R
Type outlook.exe  /safe
Disable add ins
Restart Outlook normally
