Scenario
Outlook not sending or receiving emails

What the user reports
“I’m not getting any new emails and the send button is greyed out.”

Steps taken
Check internet connection
Confirm Outlook is online
Restart Outlook
Turn Cached Mode off and on
Repair the mailbox
Create a new Outlook profile
User confirms email flow restored

Scenario
Outlook search not returning results

What the user reports
“I can’t find any emails. Search shows nothing.”

Steps taken
Open Indexing Options
Rebuild the index
Restart Outlook
Search results return normally

Scenario
Outlook stuck on “Updating inbox” for several minutes

What the user reports
“My Outlook keeps saying updating inbox and nothing loads.”

Steps taken
Check connection status
Open File then Options then Advanced
Turn off Download Shared Folders
Restart Outlook
Rebuild the OST file
Create a new Outlook profile
Issue resolved after profile recreation

Scenario
Shared mailbox calendar not updating for some users

What the user reports
“Events added by my colleague don’t show up on my calendar.”

Steps taken
Confirm user has correct permissions in Admin Centre
Remove shared mailbox from Outlook
Re add mailbox using AutoMapping
Clear Outlook cache
Calendar updates correctly

Scenario
Outlook repeatedly prompts for password

What the user reports
“I enter my password but Outlook keeps asking again.”

Steps taken
Clear Windows Credential Manager
Check for outdated authentication methods
Confirm modern authentication is enabled
Restart Outlook
User signs in successfully
